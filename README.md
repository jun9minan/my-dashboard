# 통합 대시보드

다이어리와 가계부를 한 번에 컨트롤하는 통합 대시보드입니다.

## 기능

- 🌤️ 날씨 (실시간)
- ✅ To do list (할 일 관리)
- 📝 메모 (작성 및 저장)
- 📅 일정 (오늘/내일 일정)
- 📈 미국 증시 (전일 종가)
- 💰 재무 현황 (수입/지출 관리)

## 특징

- 🎨 **미니멀 디자인**: 깔끔하고 직관적인 UI
- 🌙 **다크 모드**: 다크/라이트 모드 지원
- 📱 **반응형**: 모바일, 태블릿, PC 모두 지원
- 💾 **데이터 동기화**: Firebase로 멀티 디바이스 동기화 (준비 중)

## 사용 방법

1. `index.html`을 웹 브라우저에서 열기
2. 상단 우측의 🌙 버튼으로 다크/라이트 모드 전환
3. 각 섹션의 "상세" 또는 "전체보기" 버튼으로 상세 내용 확인

## 파일 구조

```
my-dashboard/
├── index.html    # 메인 페이지
├── style.css     # 스타일링
├── script.js     # JavaScript (다크모드 토글)
└── README.md     # 문서
```

## 향후 계획

- Firebase 연동 (실시간 데이터 동기화)
- 실제 API 연동 (날씨, 증시, 캘린더)
- 게시글 작성/수정/삭제 기능
- 멀티 사용자 지원
